# 订单设计进度管理系统

一个简洁高效的订单设计进度管理工具，支持工程师进度录入、计算预警、项目信息管理和系统配置。

## 功能特点

- **进度录入**：选择订单号、工程师和进度类型进行进度更新
- **计算与预警**：自动计算进度并生成预警信息
- **项目信息**：录入和管理订单信息
- **管理模块**：工程师管理、订单管理、进度类型管理、产品类型管理、双重确认管理

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- localStorage 数据持久化
- 响应式设计

## 使用方法

直接打开 `index.html` 即可使用。
